# Recruiting Radar UI Guide

This app is designed around one primary question:

- How far from campus are teams recruiting, and how has that changed?

## Primary View: Box Plot

The box plot is the core dashboard view.

- Each row is one team.
- Gold = current class in your selected year window.
- Blue = prior classes in the same window.
- Story badges above the chart call out the most local footprint, widest current reach, and largest current class.
- Median marker and label show the current class typical distance.
- Box = middle 50% of recruits (IQR).
- Whiskers/outliers = spread and extremes.
- Reference guides mark the local, national, and long-haul thresholds when enabled.

Use this view first for team-to-team comparison.

## Supplemental Views

Use the tabs under the main box plot for drilldowns.

- Histogram (Supplemental): distribution shape and tails.
- QA / QC Table: player-level verification details.
- How To Read This: built-in interpretation notes.

## KPI Definitions

Top stat cards summarize the filtered dataset.

- Recruits: count of included records.
- Median Miles: middle distance value.
- Avg Miles: arithmetic mean distance.
- Local Share: percent within 300 miles.
- National Share: percent beyond 1,000 miles.
- Long-Haul Share: percent beyond 2,500 miles.

## Recommended User Flow

1. Start with one team (default Arizona).
2. Add 2 to 5 comparison teams.
3. Adjust year range.
4. Read KPI cards, then compare medians and spread in the box plot.
5. Use Histogram and QA/QC only for deeper inspection.

## Data Notes

- Data source: local CSV artifacts generated by scrape_recruits.R.
- Distances: haversine (straight-line) between hometown and campus coordinates.
- Logos: local static files mapped in team_logo_urls.csv.
